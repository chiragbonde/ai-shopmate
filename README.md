# 🛒 AI ShopMate – AI Shopping Assistant

AI ShopMate is a modern AI-powered shopping assistant that helps users discover products, compare options, receive personalized recommendations, and enjoy a smarter online shopping experience.

## 🚀 Features

* 🤖 AI-powered shopping recommendations
* 🔍 Smart product search
* 🎤 Voice-based product discovery
* 🖼️ Image-based product search
* ❤️ Wishlist management
* 🛒 Shopping cart functionality
* 📊 Product comparison
* ⭐ Ratings & reviews
* 📱 Responsive modern UI
* 🌙 Dark mode support

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/ai-shopmate.git
```

Navigate to the project:

```bash
cd ai-shopmate
```

Install dependencies:

```bash
npm install --legacy-peer-deps
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
